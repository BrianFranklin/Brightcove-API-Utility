Welcome

This tool will allow you to test the Brightcove Media Read API methods.
On the top left of this page, you"ll notice a drop-down menu containing API methods grouped by which media type they relate to; videos or playlists. To get started, select the method you would like to try out. You'll need your Brightcove read API token handy, as you'll need to enter it in the resulting form.
Choosing methods causes forms on the page to dynamically update, making it clear which parameters can, and need, to be specified. As the values of these parameters are altered, the results of the API call will appear in a formatted tree view in the "return" box to the right, and the URL used to get that data will appear in the header of the page.
The tree view allows you to expand and collapse sections of the return in order to traverse it easily and inspect the data returned. This view can also be toggled to show the exact return data by clicking the "raw" link in the header of the return box.
